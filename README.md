# 企业知识库智能问答系统

基于 RAG（检索增强生成）技术构建的企业内部知识库问答系统，支持自然语言查询企业政策、规章制度等文档内容。

## 项目背景

企业内部文档分散、查找困难，员工需要频繁向HR或相关部门咨询政策问题，效率低下。本项目通过RAG技术构建智能问答系统，员工可以用自然语言直接提问，系统自动检索相关文档并给出准确回答。

## 技术架构
```
用户提问
    ↓
查询路由（判断问题类型）
    ↓
查询重写（优化检索词）
    ↓
混合检索（向量检索 + BM25关键词检索）
    ↓
RRF重排（融合两路检索结果）
    ↓
父子文档策略（小块检索 + 完整文档生成）
    ↓
LLM生成回答（GLM-4-Flash）
```

## 核心功能

- **混合检索**：结合语义向量检索和BM25关键词检索，用RRF算法融合结果，召回率优于单一检索方式
- **父子文本块**：检索用小块保证精准度，生成回答用完整文档保证上下文完整性
- **查询路由**：自动识别问题类型，分配不同处理策略
- **查询重写**：将口语化问题改写为更适合检索的形式，提升检索准确率
- **元数据过滤**：根据问题关键词自动识别文档分类，缩小检索范围
- **流式输出**：逐字输出回答，提升用户体验

## 技术栈

| 组件 | 技术选型 |
|------|---------|
| 框架 | LangChain |
| 嵌入模型 | BAAI/bge-small-zh-v1.5 |
| 向量数据库 | FAISS |
| 关键词检索 | BM25 |
| 大语言模型 | GLM-4-Flash（智谱AI） |
| 开发语言 | Python 3.12 |

## 知识库内容

系统内置5份企业文档，覆盖三个分类：

| 分类 | 文档 |
|------|------|
| HR人事 | 员工手册、薪酬福利制度、劳动合同与法务规范 |
| 财务 | 报销与差旅政策 |
| IT技术 | IT系统使用规范 |

## 项目结构
```
enterprise-rag/
├── config.py                    # 配置管理
├── main.py                      # 主程序入口
├── data/                        # 知识库文档
│   ├── hr/                      # HR人事文档
│   ├── finance/                 # 财务文档
│   └── it/                      # IT技术文档
└── rag_modules/                 # 核心模块
    ├── data_preparation.py      # 数据准备（加载、分块、元数据）
    ├── index_construction.py    # 索引构建（向量化、FAISS）
    ├── retrieval.py             # 检索优化（混合检索、RRF重排）
    └── generation.py            # 生成集成（路由、重写、回答）
```

## 快速开始

**1. 克隆项目**
```bash
git clone https://github.com/huohudas/enterprise-rag.git
cd enterprise-rag
```

**2. 安装依赖**
```bash
pip install langchain langchain-community langchain-huggingface langchain-openai faiss-cpu rank-bm25 python-dotenv sentence-transformers
```

**3. 配置API Key**
```bash
echo "ZHIPU_API_KEY=你的智谱API Key" > .env
```

**4. 运行系统**
```bash
python main.py
```

## 问答示例
```
问题：年假怎么申请？
回答：年假需提前3个工作日通过OA系统提交申请，经直属上级审批后方可休假。
      未使用年假可在次年3月31日前补休，逾期按日工资150%补偿。

问题：报销需要几天内提交？
回答：差旅费须在出差结束后7个工作日内提交报销申请。
      日常费用须在发生后30天内提交。

问题：密码需要多久更换一次？
回答：密码每90天强制更换一次，且不能重复使用最近5次的密码。
```
