# 企业知识库智能问答系统

基于 RAG 技术构建的企业内部知识库问答系统，支持员工用自然语言查询企业政策、规章制度等文档内容，解决企业内部文档分散、查找低效的问题。

## 项目背景

企业员工频繁需要查询HR政策、报销规定、IT规范等内部制度，传统方式依赖人工解答或手动翻阅文档，效率低且容易出错。本项目通过RAG技术构建统一的智能问答入口，实现自然语言直接查询。

## 系统架构
```
用户提问
    ↓
查询路由 → 识别问题类型（detail/general）
    ↓
查询重写 → 口语化问题转换为检索友好形式
    ↓
混合检索 → 向量检索（语义）+ BM25（关键词）并行
    ↓
RRF重排  → 融合两路检索结果
    ↓
父子文档 → 小块检索定位 + 完整文档生成
    ↓
LLM回答  → GLM-4-Flash 严格基于文档内容生成
```

## 核心技术

**混合检索 + RRF重排**
单纯向量检索对专有名词存在盲区，BM25关键词检索补充精确匹配能力。两路结果通过RRF算法融合，在两个检索里同时排名靠前的文档得分最高，召回率优于单一检索方式。

**父子文本块策略**
文档按Markdown标题切分为小块用于检索，检索命中后向上获取完整父文档传给LLM。解决了精准检索与完整上下文的矛盾，避免因分块截断导致回答缺失关键信息。

**查询路由 + 查询重写**
路由层自动判断问题类型，分配不同处理策略。重写层将"我想请假"改写为"员工请假申请流程和审批规定"，提升检索准确率。

**RAGAS量化评测**
构建覆盖HR、财务、IT三领域的30道测试问题集，使用RAGAS框架持续监控系统质量，建立可量化的评估机制。

## 评测结果

| 指标 | 得分 | 基准 | 说明 |
|------|------|------|------|
| Faithfulness（忠实度）| 0.40 | >0.80 | 受免费模型指令遵循能力限制，生产环境换用更强模型可提升至0.8+ |
| ContextRecall（上下文召回）| 0.80 | >0.70 | 检索质量良好，80%的标准答案能在检索文档中找到 |

## 技术栈

| 组件 | 选型 | 选型理由 |
|------|------|---------|
| 框架 | LangChain | 模块化设计，各组件可独立替换 |
| 嵌入模型 | BAAI/bge-small-zh-v1.5 | 中文优化，本地运行无API费用 |
| 向量数据库 | FAISS | 轻量本地存储，支持持久化 |
| 关键词检索 | BM25 | 弥补向量检索的专有名词盲区 |
| 大语言模型 | GLM-4-Flash | 免费额度充足，适合开发测试 |
| 评测框架 | RAGAS 0.4.3 | RAG领域主流评测框架 |

## 知识库

| 分类 | 文档 | 内容概要 |
|------|------|---------|
| HR人事 | 员工手册 | 考勤、假期、行为规范、离职流程 |
| HR人事 | 薪酬福利制度 | 薪资构成、绩效奖金、五险一金、员工福利 |
| HR人事 | 劳动合同与法务规范 | 合同管理、经济补偿、竞业限制 |
| 财务 | 报销与差旅政策 | 费用报销标准、差旅审批、违规处理 |
| IT技术 | IT系统使用规范 | 账号密码、数据安全、违规处理 |

## 项目结构
```
enterprise-rag/
├── config.py                    # 集中管理所有配置参数
├── main.py                      # 主程序，串联四个核心模块
├── data/                        # 知识库文档
│   ├── hr/
│   ├── finance/
│   └── it/
├── evaluation/
│   ├── evaluate.py              # RAGAS评测脚本
│   ├── test_questions.md        # 30道测试问题集
│   └── ragas_scores.json        # 评测结果
└── rag_modules/
    ├── data_preparation.py      # 文档加载、分块、元数据提取
    ├── index_construction.py    # 向量化、FAISS索引构建与持久化
    ├── retrieval.py             # 混合检索、RRF重排、元数据过滤
    └── generation.py            # 查询路由、重写、LLM回答生成
```

## 快速开始
```bash
git clone https://github.com/huohudas/enterprise-rag.git
cd enterprise-rag
pip install langchain langchain-community langchain-huggingface langchain-openai faiss-cpu rank-bm25 python-dotenv sentence-transformers ragas openai
echo "ZHIPU_API_KEY=你的Key" > .env
python main.py
```
